dg-platform-agent has been deployed with the following components:

  1. Controller  — manages ephemeral agent pods
  2. WS Gateway  — WebSocket bridge (Redis pub/sub)
  3. Agent Pods   — created dynamically by the Controller (not deployed by this chart)
